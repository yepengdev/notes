#+TITLE: 读完一章小说需要考虑的东西
#+AUTHOR: 阅读实验室
#+DATE: 2026-04-21
#+OPTIONS: toc:3 num:nil

对于类型小说，这种框架只适合第一章、重点转折章和高潮章节。其他以详细到场景的梗概代替。

+ 沉浸阅读 :: 不做笔记，纯粹享受故事。
+ 第一反应 :: 读完一章后，合上书停顿 =30 秒= ，让情绪余韵沉淀。
+ 情感复盘 :: 先回答 /📖 读者板块/ 的问题（捕捉直觉）。
+ 技术拆解 :: 切换至/作者板块/，像建筑师看承重墙一样审视文本。
+ 迁移应用 :: 记录可学习的技巧。
+ [ ] 停顿 30-60 秒，不要立刻翻页。
+ [ ] 记录 /第一反应词/（例如：爽快、压抑、困惑、温暖）。
+ [ ] 一句话梗概：用一句话写清这一章的客观事件。
+ [ ] 核心转折：这一章里最重要的单一事件是什么？
+ [ ] 变化清单：相比章节开头，人物处境或关系发生了什么具体变化？
+ [ ] 意外元素：有没有让你感到惊讶的情节发展？
+ [ ] 读完这章，我对某个角色的看法改变了吗？/为什么？/（是哪句台词或哪个行动导致的？）
+ [ ] 此刻，主人公 =想要= 什么？（短期目标）与 =需要= 什么？（深层成长）有冲突吗？
+ [ ] 站队时刻：在人物冲突中，我站谁？作者让我理解对立面了吗？
+ [ ] 信息释放：关于角色的背景信息，是通过以下哪种方式透露的？
  - 动作描写？
  - 对话潜台词？
  - 内心独白？
  - 其他角色的评价？
+ [ ] POV 合理性：为什么这一章必须由 =这个角色= 的视角来讲述？他的叙述带有什么偏见？
+ [ ] 情绪基调：这一章的主导情绪是什么？（窒息感？释然？偏执？喜悦？）
+ [ ] 意象扫描：有没有反复出现的物品、颜色或天气？
+ [ ] 金句摘抄：抄下 1-3 句你觉得优美、深刻或极其巧妙的句子。
  #+BEGIN_EXAMPLE
  蓝色高亮： 文笔优美
  粉色高亮： 人物刻画
  黄色高亮： 伏笔线索
  #+END_EXAMPLE
+ [ ] 存在理由：这一章在全书中的叙事功能是什么？
  - [ ] 推进情节？
  - [ ] 揭示人物？
  - [ ] 建立世界观？
  - [ ] 调节节奏（缓冲）？
+ [ ] 删除测试：如果删掉这一章，故事还讲得通吗？如果不通，缺失的关键信息是什么？
+ [ ] 结尾逻辑：作者为什么选择 =在这个瞬间= 结束这一章？（悬念？意象？情感转换？转折？）
+ [ ] 行动 vs. 反应：这一章是“做事遇阻”的场景，还是“休整消化”的场景？
+ [ ] 时间流速：这一章覆盖了多久的时间线？（十分钟的细节 vs. 三周的概括）
+ [ ] 节奏成因：这一章 /感觉上/ 是快还是慢？是什么造成了这种感觉？（短句、对话量、描写密度）
+ [ ] 第一句话：第一句起了什么作用？（制造悬念？确立氛围？直接动作？）
+ [ ] 感官细节：除了视觉和听觉，作者有没有用到 嗅觉、触觉、温度 来让世界真实？
+ [ ] 注意力审计：我在哪里走神了？为什么？（信息堆砌？对话打转？描写冗长？）
+ [ ] 阅读动力：是什么具体的张力推着我一直翻页？（想知道某个秘密？担心角色安危？）
+ [ ] 冲突性质：人物对抗的是外部力量（反派、风暴）还是内部力量（怀疑、悲伤）？
+ [ ] 隐瞒的艺术：作者此时此刻仍然 /没有/ 告诉我们什么？这种隐瞒是刻意为之吗？
+ [ ] 对下文的预测：基于这章的结尾，我认为接下来会发生什么？
+ [ ] 外部连接：这一章让我想起了另一本书、一部电影或自己的生活经历吗？
+ [ ] 技能迁移：这一章里有一个具体的技巧，我可以直接 =偷用= 在我自己的写作里吗？

#+BEGIN_QUOTE
“作者为什么选择 /在这里/ 结束这一章？”
#+END_QUOTE

这一份是轻量级勾选清单，适合读完一章后花 =5分钟= 快速完成：

- [ ] 暂停：合上书，停顿了 30 秒。
- [ ] 情节：用一句话概括了本章事件。
- [ ] 情感：记录了一个具体的情感变化瞬间。
- [ ] 结构：分析了第一句话或最后一句话的功能。
- [ ] 技法：找出了一个“可偷用”的写作手法。
- [ ] 结尾：回答了“为什么结束在这里？”
