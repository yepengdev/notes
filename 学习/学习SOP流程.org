* 1 预习（时间 = 视频时长 × 5%）

- [ ] 扫标题、描述、章节标记 :: 30s
- [ ] 拖进度条感知结构 :: 1min
- [ ] 激活旧知：写下"关于这个主题，我已经知道什么？" :: 2min
- [ ] 设定 2~3 个具体问题（不能是/否回答，须涉及关系/因果/机制） :: 1min
  - 格式：=X 和 Y 的关系是什么？= / =为什么 Z 成立？=
- [ ] 设定最低目标："这节课结束后，我最低要能_______" :: 30s

* 2 观看循环（重复至结束，每循环 3~8min，推导密集处 2min）

** 2.1 观看

1.25× 速，以下信号立刻暂停：

| 信号                  | 动作           |
|--------------------|--------------|
| "换句话说…"           | 先用自己的话说      |
| 老师提一个问题           | 先自己尝试回答      |
| 你感到"等等，为什么？"      | 写下困惑         |
| 老师开始推导            | 预判下一步        |
| "关键在于…"           | 高度集中         |
| 老师展示结论/解法/推导完成前    | 暂停，自己预测      |

** 2.2 闭眼默想（30s）

- 前 10s：脑内回放核心内容
- 中 10s：一句话概括——最重要的一件事
- 後 10s：我真的理解了，还是只是认出了那些话？
  - *测试：我能不能用完全不同的词说同一件事？*

** 2.3 锚点笔记（1~2min）

#+BEGIN_EXAMPLE
⏱ 3:42  【核心概念】一句话
          ↓ 为什么重要 / 逻辑流向
          ❓ 疑问（不试图当场解决）
#+END_EXAMPLE

* 3 节后重建（每大节结束，5~10min）

- [ ] 关闭视频 + 合上笔记
- [ ] 白纸画本节概念骨架（3~5min）
  - 箭头 = 依赖关系（A → B 表示"要理解 B 必须先理解 A"）
  - *不是思维导图*
- [ ] 口头叙述本节核心（1~2min）——*大声说出来*
- [ ] 红笔标注模糊点（1min）——*不要立刻回查*

* 4 全局整合（视频结束后，15~30min）

- [ ] 绘制全局概念图（5~10min）
  - 大纸，聚合各节骨架
  - 绿箭 = 顺承 | 红箭 = 矛盾/张力 | 蓝箭 = 类比
- [ ] 回答预习问题（5min）——答不出则入疑问清单，标 ⭐⭐⭐
- [ ] 费曼式总结（10min，500~800 字）
  1. 这节课在解决什么问题？（一句话）
  2. 核心直觉是什么？（一个比喻或图像）
  3. 从直觉到结论，关键逻辑步骤？
  4. 只说一件事让听众带走，那是什么？
  5. 限制/边界条件/例外？
- [ ] 整理未解决疑问清单（5min）
  | 疑问 | 当前理解 | 为何困惑 | 优先级 |
  ⭐⭐⭐ 阻碍后续 | ⭐⭐ 局部困惑 | ⭐ 好奇追问

* 5 产出层（全局整合后，30~60min）【仅核心视频完整执行】

:ID:       5
:END:

** 5A 刻意练习（25~45min）

- [ ] 选题：基础 ×2 + 中档 ×2 + 挑战 ×1
- [ ] 严格计时完成
- [ ] 对照答案，标错因类型：
  - ☐ 概念错误 ☐ 适用条件错误 ☐ 计算/执行错误 ☐ 遗漏 ☐ 策略错误
- [ ] 每道错题：不看答案重做 + 写一句诊断 =我犯错是因为…=
- [ ] 录入错误日志

** 5B 生成性练习（15~20min）

- [ ] 变式问题："改变一个前提条件，结论会怎样？构造反例。"
- [ ] 边界案例："什么极端情况下这结论失效？构造临界例子。"
- [ ] 跨域类比："这概念让我联想到_____。似？异？"
- [ ] 出题："如果我是教授，出什么题区分真懂和假懂？"

* 6 社会校准（每周 1~2 次）

** 6A 费曼教学（30min，≥1 次/周）

- [ ] 找真人（或录音後隔天自听）
- [ ] 不借助笔记，口头讲授本周核心
- [ ] 听众在任何"听不懂"处打断
- [ ] 被打断时：换方式/换类比——*不能用术语解释术语*
- [ ] 记录所有被打断点 → 这些是你的理解裂缝

** 6B 同伴讨论（45~60min，1 次/周）

- [ ] 各自准备本周最困惑的 1~2 个问题（5min）
- [ ] A 讲解 B 追问（15min）→ 互换（15min）
- [ ] 对比概念图（10min）
  - 为什么你记了这个而我没记？
  - 哪些你认为重要但我完全忽略了？
- [ ] 联合解一道难题（剩余时间）

* 7 后续巩固

** 7A 24h 内（10min）

- [ ] 翻看锚点笔记和概念图（5min）
- [ ] 合上一切，白纸提取（5min）
  - 写出所有核心概念 + 默画概念图 → 对照，标遗漏

** 7B 1 周后（20~30min）

- [ ] 白纸重建全局概念图（不看原图）
- [ ] 写出关键推导逻辑链
- [ ] 尝试回答 24h 时的疑问——现在能答了吗？
- [ ] 对照原图，标遗忘区（全部重建完再对照）
- [ ] 回顾本周错误日志——有反复出现的错因吗？有则找同类题加练

** 7C 1 个月后（15min）

- [ ] 快速浏览概念图（3min）
- [ ] 口头费曼总结（5min）
- [ ] 跨域迁移尝试（5min）：这概念现在能和哪些其他领域知识建立联系？
- [ ] 发现大片遗忘 → 说明 1w 重建不够扎实

* 8 跨周整合（每 2~3 周一次）

** 8A 交叉练习（30~45min）

- [ ] 从过去 3~5 个主题各抽 2~3 道题 → 打乱，不标来源
- [ ] 计时完成，每题第一步：判类型
- [ ] 分析：哪些题判错类型？容易混淆的概念是？

** 8B 跨域迁移（30min）

- [ ] 选两个不同领域的本周核心概念
- [ ] 找深层结构相似性 → 写一个"迁移尝试"
- [ ] 评估：深刻还是牵强？为什么？（牵强也是训练）

** 8C 系列整合（学完整系列后）

- [ ] 拼接所有全局概念图
- [ ] 找共同主题、反复出现的核心思想
- [ ] 找跨课程的矛盾或张力
- [ ] 写出该领域的"个人知识地图"

* 领域适配速查

| 领域             | 强化                                   | 替换                                     |
|-----------------+---------------------------------------+-----------------------------------------|
| 数学/物理/理论 CS | 5A 练习量加倍（10 道）；加证明构造+逆向训练 | —                                       |
| 文学分析/哲学     | 5B 多视角阐释；6B 论证批评               | —                                       |
| 写作             | 缩减至 1~4 阶段                         | 5A → 写作循环（分析→模仿→创作→冷却→修改×3）  |
| 编程             | 完整 SOP（仅概念部分）                   | 5A → 编程循环（跟练→变式→从零构建→调试→读码） |
| 语言学           | + 数据操作（拿真实语料独立分析）           | —                                       |
| 历史             | + 一手史料接触 + 反事实推理               | —                                       |

* 月度自检

- [ ] 24h 提取正确率 > 80%
- [ ] 1w 白纸重建完整度 > 70%
- [ ] 1mo 核心节点保持度 > 90%
- [ ] 本周我创造了什么？（非笔记的新东西）
- [ ] 本周最典型的 3 个错误类型
- [ ] 本周至少 1 次被他人指出"你其实没懂"
- [ ] 本周发现 ≥1 个跨领域概念联系
- [ ] 本周提出 ≥1 个自己回答不了但值得探索的新问题

* 故障排查

| 症状             | 诊断               | 调整                |
|-----------------+-------------------+--------------------|
| 24h 提取差       | 节後重建不够深       | 延长重建，强制口头叙述 |
| 1w 大片遗忘       | 24h 只是翻看而非提取 | 强制合上再回忆        |
| 能说但做不出题     | 缺 5A 刻意练习      | 加量 + 严格计时      |
| 总是同类错误      | 错误分析不系统       | 回顾错误日志，找模式   |
| "看了很多说不出"   | 观看时没暂停预测     | 增加暂停频率         |
| 概念图很散        | 全局整合被跳过       | 至少留 15min 画跨节  |
| "感觉都懂但没进步" | 缺外部校准          | 引入同伴讨论/费曼教学  |
| 学了很多但产出少   | 跳过 5 产出层       | 5A 5B 不可省略       |
